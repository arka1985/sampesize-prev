Sample Size Calculator

For Prevalence Epidemiological Studies

Sample Size (N) 0 N = 4PQ / D²
P = 0.50 Q = 0.50 D = 0.05

The "P" represents the expected prevalence (proportion) of the disease or condition in the population (0-100%).

The "D" is the absolute error or precision you are willing to accept (typically 5%). Smaller D means higher precision but requires larger N.

Formula Steps

1. N = Sample Size

2. P = Prevalence (as decimal)

3. Q = 1 - P

4. D = Precision (as decimal)

5. 4 = Constant for 95% Confidence (approx 1.96² ≈ 4)

Interpretation

This formula determines the minimum number of participants needed to estimate a population prevalence with a specified level of precision and confidence.
